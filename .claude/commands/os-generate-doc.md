# Génération Documentation Technique

Génère doc markdown dans `docs/features/`

## Usage

```
/os-generate-doc {feature_name}
```

## Structure

```markdown
# {Feature Name}

## Context Métier
[Pourquoi cette feature]

## Fonctionnement
[Description haut niveau]

## Architecture
[Composants principaux]

## Configuration
[Variables env, config Symfony]

## Utilisation
[Exemples API, CLI]

## Troubleshooting
[Problèmes fréquents + solutions]

## FAQ
[Questions fréquentes]
```
